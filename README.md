# timer-app

Timer app with history of your tasks!

## TOOLS

- create-react-app
- React.js
- JS Locale Storage
- styled-components
- sass/scss

## Available scripts

`npm start` - runs development mode

`npm run build` - runs build process for production

`npm run deploy` - runs build process and publish the page using `gh-pages` branch
